# SMO_Randomizer_Tracker

A live tracker + OBS overlay for Super Mario Odyssey Ranomizer speedruns.

## Requirements
- Python 3.11+  
Download from: https://www.python.org/downloads/
✔ Make sure "Add Python to PATH" is checked

## Installation
1. Download and unzip
2. Open a terminal in the project folder
3. Install dependencies: pip install -r requirements.txt
4. Run the tracker: python tracker.py

## Note: If you don't have python or just want to run the tracker easier, double click the .exe file (Made with Pyinstaller and has all dependencies baked in)
